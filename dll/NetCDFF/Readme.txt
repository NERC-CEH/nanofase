NetCDF 4.4 built for Win32 using IFort 14.0.4. The following routines ware not available in the sources and were replaced with dummy code: _nc_create_par_fortran, _nc_open_par_fortran, _nc_var_par_access, _ncerr and  _ncopts.

The source code, .inc and .h files, if needed, are to be unpacked from the distribution file netcdf-fortran-4.4.0.tar.gz from UCAR.
